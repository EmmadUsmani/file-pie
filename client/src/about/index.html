File Pie

Hi, I'm Emmad. I created File Pie out of frustration with the friction in sending files, both to my own devices and to other people. I was tired of compressing files to send via email, only to get blocked by malware filters, and uploading files to privately operated cloud storage platforms, having to manually delete them later.


File Pie simplifies this process: simply upload a file then share a link or QR code with the recipient.

Privacy

File Pie works using WebRTC, a browser protocal enabling peer to peer connections. This means that files are transferred directly to recipients, without ever being sent to a server. A signaling server is used to establish a connection between browsers, but during this process no information about the file is sent to the server. The app is completely open source.

Inspiration

Two similar projects, File Pizza and Sharedrop served as inspirations for File Pie. File Pizza in particular served as an inspiration for the design and name. Code from both projects was referenced during development, but no source code was used directly.
